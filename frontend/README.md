# Scroll-Driven Hero Animation ##(increase your brightness for better glow)

## Tech Stack
- React (Vite)
- Tailwind CSS
- GSAP (ScrollTrigger)

## Features
- Smooth scroll-based animation tied to scroll position
- Staggered text reveal on load
- Optimized performance (minimal reflows, transform-based animation)
- Responsive layout

## Focus
Built with emphasis on:
- Smoothness
- Performance
- Clean animation architecture
